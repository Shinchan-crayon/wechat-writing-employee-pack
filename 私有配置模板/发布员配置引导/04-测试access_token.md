# 测试平台 API 连通性

## 这一步干什么

验证你的平台凭证和网络权限配置是否正确，能否正常调用平台 API。

## 测试方法

向平台提供的 token 获取接口发送请求。具体接口地址和参数参见平台官方 API 文档。

## 成功的样子

返回包含有效凭证和过期时间的 JSON 响应。

## 常见失败

- **凭证无效** → 检查平台后台的凭证配置是否正确
- **IP 不在白名单** → 确认当前机器出口 IP 已加入平台 IP 白名单
- **网络不通** → 检查是否开了代理/VPN 干扰

## 通过标准

能成功获取有效凭证即为通过。
