# 常见错误排查

## 错误类型速查

| 错误类型 | 含义 | 检查方向 |
|:------|------|---------|
| 凭证无效 | 开发者密钥错误 | 回平台后台重置密钥，复制新的 |
| 网络地址不在允许列表 | IP 被拒绝 | 查询当前出口 IP → 添加到平台允许列表 |
| 凭证过期 | 临时凭证超时 | 重新获取凭证（通常有效期 2 小时） |
| 调用频率超限 | 请求太频繁 | 等 1 分钟后重试 |
| 无 API 权限 | 账号未完成认证 | 检查账号认证状态 |
| 内容长度超限 | 正文超过平台限制 | 分段处理 |
| 媒体引用无效 | 图片未上传或引用已过期 | 重新上传图片获取新的引用 ID |

## 常见场景排查

### 场景 1：昨天还能用，今天不行了

**最可能的原因：网络出口地址变了。**
查询当前出口地址，对比平台后台允许列表。如果不一致 → 地址变了，重新添加。

**次可能原因：凭证过期。**
平台 API 凭证通常有有效期（如 2 小时）。发布员在调用 API 前会自动续期，但如果中间网络中断太久可能导致凭证实际过期。重试一次通常就恢复。

### 场景 2：草稿创建成功但图片不显示

平台草稿 API 的图片需要先通过素材上传接口上传，拿到引用 ID 后再在 HTML 中引用。

检查流程：
1. HTML 中的图片引用格式是否符合平台要求
2. 是否先调了素材上传接口上传图片
3. 草稿内容中图片引用格式是否正确

### 场景 3：标题/正文推送到草稿箱后乱码

**原因：** HTML 中有平台不支持的标签或样式。

平台草稿箱支持的标签有限。发布员推送前会自动清理不支持的标签，但如果你的 HTML 用了复杂的 CSS 内联样式，部分可能会丢失。推送后的人工检查步骤就是用来发现这类问题的。

## 还是搞不定

1. 重新走一遍连通性测试，确认基础链路通畅
2. 查阅平台官方 API 文档
3. 如果浏览器操控方式能正常工作但 API 方式不行 → 暂时切到浏览器方式，API 排查完再切回来

```yaml
# USER.md 中切换模式
publisher:
  publish:
    method: "browser"    # 从 api 切换到 browser
```
