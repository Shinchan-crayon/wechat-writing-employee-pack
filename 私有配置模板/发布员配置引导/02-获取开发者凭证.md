# 获取微信开发者凭证

## 你需要什么

微信开发者平台中的 AppID 和 AppSecret。

⚠️ **开发者密钥等于你的账号密码。** 泄露后别人可以替你发文章、群发消息、获取用户数据。不要截图、不要发到聊天里、不要提交到 git。

## 在哪获取

1. 登录微信公众平台或微信开发者平台
2. 进入「设置与开发」→「基本配置」或类似页面
3. 找到开发者凭证区域，复制 AppID 和 AppSecret
4. 将凭证写入 `USER.md`：

```yaml
publisher:
  platform:
    app_id: "你的平台AppID"
    app_secret: "你的AppSecret"
```

## 通过标准

- `USER.md` 中 `publisher.platform.app_id` 不为空
- `USER.md` 中 `publisher.platform.app_secret` 不为空

## 常见问题

**Q：密钥忘了怎么办？**
A：回到平台开发者后台，重置密钥。旧的会立即失效，生成新的。

**Q：密钥填到 USER.md 后安全吗？**
A：USER.md 不进入公开包、不进 git。agent 读取时密钥不进入日志。只要你的电脑本身安全，这个方案比存在浏览器里更安全。
