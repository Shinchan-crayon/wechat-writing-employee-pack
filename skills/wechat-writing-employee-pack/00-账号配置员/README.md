# 账号配置员

账号配置员是公众号写作员工包的第一步，负责把后续写作流程需要的账号、工作区、风格卡和本地用户配置先准备好。

## 它解决什么问题

很多写作工作流失败，不是因为写手不会写，而是因为一开始没有确认：

- 写哪个公众号账号
- 用哪套写作风格
- 素材和成稿放在哪里
- 是否需要接入公众号草稿箱
- 凭证和私有配置应该放在哪里

账号配置员的作用就是先把这些基础信息问清楚，避免后面的员工边写边猜。

## 它负责什么

- 引导用户生成本地 `USER.md`
- 确认公众号账号、文章类型、目标读者和工作区
- 绑定或创建账号风格卡
- 询问是否启用发布员
- 如果启用发布员，引导用户准备 AppID、AppSecret 和网络允许列表

## 它不负责什么

- 不写正文
- 不做素材搜索
- 不核查文章事实
- 不把 AppSecret、access_token 或真实 `USER.md` 写进公开仓库

## 前置条件

无。第一次使用总包时应该先启动账号配置员。

## 推荐安装

账号配置员依赖总包里的共享资源、模板和后续员工协作顺序，不建议脱离总包单独安装。

```bash
npx skills add https://github.com/Shinchan-crayon/wechat-writing-employee-pack/tree/main/skills/wechat-writing-employee-pack
```

安装后对 AI 助手说：

```text
请读取公众号写作员工包，先执行账号配置员。
```
