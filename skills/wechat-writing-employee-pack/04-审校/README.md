# 审校

审校负责在文章写完之后做发布前检查，包括降 AI 味、事实表达复核、日期核实、排版检查和归档。

## 它解决什么问题

一篇 AI 辅助写出来的文章，即使事实核过、正文也写好了，发布前仍然可能有 AI 口吻、标题正文不一致、日期模糊、表达过度、排版不稳等问题。审校就是最后一道质量门。

## 它负责什么

- 降低正文 AI 味和模板感
- 做发布前 9 项自检
- 核实日期、时间线和关键表述
- 检查标题、正文、排版和风险表述
- 归档最终 HTML 稿件和审校记录

## 它不负责什么

- 不重写整篇文章结构
- 不放过未核实日期
- 不负责平台发布
- 不绕过用户最终确认

## 前置条件

- 写手已经交付 HTML 正文和标题候选
- 事实核查员已经完成写后反查

## 推荐安装

审校依赖总包里的正文、核查结果、排版规则和归档流程，不建议脱离总包单独安装。

```bash
npx skills add https://github.com/Shinchan-crayon/wechat-writing-employee-pack/tree/main/skills/wechat-writing-employee-pack
```

安装后对 AI 助手说：

```text
请读取公众号写作员工包，对这篇公众号 HTML 稿件启动审校并归档。
```
