# 注册与认证

## 你需要什么

要使用平台 API 推送草稿，你需要一个**已完成认证的平台账号**。

## 适用账号类型

| 账号类型 | 能否用 API | 说明 |
|---------|:---:|------|
| 个人账号 | ❌ | 通常无 API 权限 |
| 认证账号 | ✅ | 需完成平台认证 |

如果你是个人账号，升级为认证账号后即可使用。

## 在哪认证

1. 登录平台后台
2. 进入「设置」→「账号设置」或类似页面
3. 找到「认证」入口，按引导提交资料
4. 认证审核通常需要几个工作日

## 通过标准

平台后台账号设置页面显示「已认证」标识。

## 常见问题

**Q：我已经认证了，但后台显示未认证？**
A：可能认证已过期。检查认证有效期。

**Q：我不想花钱认证，能用浏览器操控方式吗？**
A：可以。浏览器方式不需要 API 权限——只要浏览器已登录目标平台，发布员可以通过操控浏览器来创建草稿。但这种方式稳定性不如 API，且需要浏览器保持登录状态。
