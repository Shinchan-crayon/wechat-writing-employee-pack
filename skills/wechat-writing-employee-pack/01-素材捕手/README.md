# 素材捕手

素材捕手负责把一个选题、链接、零散素材或初步想法，整理成可以进入写作的文章框架。

## 它解决什么问题

AI 直接写全文时，经常会把结构写顺，但来源不清、信息不全、重点跑偏。素材捕手先做写作前的准备工作：读素材、找补充信息、搭结构，再交给事实核查员。

## 它负责什么

- 读取账号配置和风格卡
- 读取用户提供的源素材
- 搜索补充 3-5 个可靠来源
- 提炼核心观点、受众和文章角度
- 生成文章框架和待核查信息清单

## 它不负责什么

- 不直接写完整文章
- 不跳过用户确认
- 不替代事实核查员做最终核查
- 不伪造来源

## 前置条件

- 账号配置员已经完成本地 `USER.md`
- 已确认目标账号、工作区和风格卡
- 用户提供了选题、链接、素材或明确方向

## 推荐安装

素材捕手依赖总包里的账号配置、共享资源和后续核查流程，不建议脱离总包单独安装。

```bash
npx skills add https://github.com/Shinchan-crayon/wechat-writing-employee-pack/tree/main/skills/wechat-writing-employee-pack
```

安装后对 AI 助手说：

```text
请读取公众号写作员工包，我已经完成账号配置，现在让素材捕手整理这个选题。
```
