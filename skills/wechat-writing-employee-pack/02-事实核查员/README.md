# 事实核查员

事实核查员负责检查文章里的数据、价格、融资、案例、引用、时间线和关键判断，避免公众号文章因为一个未经核实的细节翻车。

## 它解决什么问题

很多内容看起来像事实，其实只是二手转述、旧数据、平台概念混用，或者没有确认日期。事实核查员把这些风险点单独拎出来，在写前建核查表，写后逐句反查。

## 它负责什么

- 对素材捕手交付的框架做写前核查
- 标出需要确认的数据、价格、案例、引用和时间
- 优先寻找一手来源或更可靠来源
- 对成稿做写后逐句反查
- 输出事实核查表和风险提示

## 它不负责什么

- 不为了文章流畅放过未经确认的数据
- 不把二手转述当最终来源
- 不负责正文润色
- 不替用户做投资、医疗、法律等高风险结论

## 前置条件

- 素材捕手已经交付文章框架和来源列表
- 已明确哪些信息会进入正文

## 推荐安装

事实核查员依赖总包里的素材框架、参考资料和后续写手协作，不建议脱离总包单独安装。

```bash
npx skills add https://github.com/Shinchan-crayon/wechat-writing-employee-pack/tree/main/skills/wechat-writing-employee-pack
```

安装后对 AI 助手说：

```text
请读取公众号写作员工包，用事实核查员检查这份文章框架和素材来源。
```
