# 禁用词表

所有员工在生成、核查、审校阶段均需检查本文档。命中 = 重写。

## 完全禁用词（出现即不合格）

### 开头禁用
- 近日……
- 随着……
- 在当前背景下……
- 值得注意的是……
- 不容忽视的是……
- 本文将……

### AI 特征词禁用
- 赋能、助力、闭环、抓手、对齐、拉通
- 瞬间归零、瞬间蒸发、立马、迅速
- 简直离了个大离谱
- 扎心、破防
- 实现（在非必要时）
- 进行（在非必要时）
- 标志着
- 众所周知

### 胶水词限用（每篇最多 1 次）
- 说白了
- 你会发现
- 真正值得看的
- 更重要的是
- 这说明什么
- 某种程度上
- 归根结底

### 对照句型完全禁用
- 不是……而是……
- 从来不是……而是……
- 看着像……其实是……
- 这已经不是……而是……
- 如果……那……（作为对照框架时）
- 不是什么……是什么……
- 真正（如"真正麻烦的地方""真正值得看的"）
- 最难看的（如"最难看的地方是"）

### 填充/空壳句禁用
- 从某种意义上说……
- 这充分说明了……
- 市场教育了我们……
- 投资者应保持理性……
- 机遇与风险并存……
- 发展路线发生改变
- 改变发展动力
- 最优秀的业绩
- 全链路协同
- 运营驱动/创新驱动
- 营销效率优先/产品创新优先

### 赘述句式禁用
- 不是单纯……/不只是……
- 这件事放在以前……
- 换个角度看……
- 把话放得很直
- 最扎眼的是
- 原因不复杂
- 说到底
- 真正关键是
- 这才是这波最值得盯的地方
- 真正要命的是
- 公开报道里/有媒体称/据说（必须直接写来源主体）
- 看到这里就很清楚了
- 这一步很关键
- 事情开始变味

### 结构禁用
- 首先/其次/再次/最后
- 第一/第二/第三
- 背景→事件→原因→影响→结论 线性骨架

### 格式禁用
- 项目符号列表（`-`、`•`、数字编号）
- 分隔线（`---`）
- 破折号（`——`）—— 替换为逗号或断句

### 推广/营销词禁用
- 暴富 / 稳赚 / 闭眼冲 / 内幕保真

## 推荐替换方向

| 原词 | 推荐替换 |
|------|---------|
| 发展路线发生改变 | 现在要走的路变了 / 现在押的重点变了 |
| 改变发展动力 | 换增长抓手 / 换主导能力 |
| 最优秀的业绩 | 最能打的一仗 / 最硬的一次 |
| 全链路协同 | 硬件、软件、生态一起动 |
| 标志着…… | 看到这里就知道…… |
| 运营驱动/创新驱动 | 靠运营带着跑 / 靠产品和创新往前顶 |

## 过渡句规则

过渡句如果不提供新时间、地点、人物、数字、动作、结果，默认删掉。每段开头优先直接给事实，不先铺情绪判断。
